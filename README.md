Arduino pins:

I2C bus: A4, A5
  - AHT10 temp/humidity sensor
  - DS 3231 real time clock
  - KC-1602-BB-I2C LCD display

Potentiometer: A3

Resistor ladder buttons: A2

NTC thermistor: A1


Wifi (currently unused): D0, D1

Relay module: D2, D3, D4, D5

SD card/SPI (currently unused): D10, D11, D12, D13
